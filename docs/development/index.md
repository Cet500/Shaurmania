# Руководство для разработчиков

## Обзор

Этот раздел содержит подробное руководство по разработке в проекте Shaurmania.

Здесь вы найдете информацию о стиле кода, работе с моделями, представлениями,
тестировании, оптимизации и отладке.

## Структура раздела

- **[Начало работы](getting-started.md)** — настройка окружения, структура проекта
- **[Стиль кода](code-style.md)** — конвенции и стандарты кодирования
- **[Работа с моделями](models.md)** — создание моделей, валидация, оптимизация
- **[Работа с представлениями](views.md)** — views, формы, обработка ошибок
- **[Тестирование](testing.md)** — pytest, Factory Boy, покрытие кода
- **[Оптимизация](optimization.md)** — оптимизация запросов, кэширование, производительность
- **[Отладка](debugging.md)** — инструменты отладки, логирование, решение проблем

## Быстрый старт

1. Прочитайте [Начало работы](getting-started.md) для настройки окружения
2. Изучите [Стиль кода](code-style.md) для понимания конвенций проекта
3. Используйте соответствующие разделы при работе над конкретными задачами
