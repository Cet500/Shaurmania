# Приложение `main`

## Назначение

Основное приложение проекта Shaurmania, содержащее ключевую бизнес-логику:
пользователи, товары, отзывы, локации, достижения, новости и акции.

## Основные функции

- **Управление пользователями** — кастомная модель пользователя с расширенными полями
- **Каталог товаров** — шаурма с категориями, изображениями, описаниями
- **Система отзывов** — отзывы и рейтинги товаров
- **Локации доставки** — точки выдачи и доставки
- **Достижения** — система геймификации
- **Новости** — публикация новостей с тегами
- **Акции** — управление скидками и акциями

## URL-маршруты

| URL                | Представление | Описание             |
|--------------------|---------------|----------------------|
| `/`                | `index`       | Главная страница     |
| `/catalog`         | `catalog`     | Каталог товаров      |
| `/product/<slug>`  | `product`     | Страница товара      |
| `/search`          | `search`      | Поиск                |
| `/login`           | `login`       | Вход                 |
| `/reg`             | `reg`         | Регистрация          |
| `/logout`          | `logout`      | Выход                |
| `/user/<username>` | `user`        | Профиль пользователя |
| `/locations`       | `locations`   | Список локаций       |
| `/location/<slug>` | `location`    | Страница локации     |
| `/news`            | `news`        | Новости              |
| `/news/<slug>`     | `news_detail` | Детальная новость    |
| `/stocks`          | `stocks`      | Акции                |
| `/stock/<slug>`    | `stock`       | Детальная акция      |
| `/about`           | `about`       | О нас                |
| `/feedback`        | `feedback`    | Отзывы               |
