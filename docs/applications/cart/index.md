# Приложение `cart`

## Назначение

Приложение для управления корзиной покупок, заказами и промокодами.

## Основные функции

- **Корзина покупок** — добавление/удаление товаров, управление количеством
- **Заказы** — история заказов пользователей
- **Промокоды** — система скидок с ограничением по времени

## Структура

```
cart/
├── models.py         # Модели Cart, Order, Promocode
├── views.py          # Представления корзины
├── urls.py           # URL-маршруты
├── admin.py          # Настройки админ-панели
└── fixtures/         # Тестовые данные (промокоды)
```

## URL-маршруты

| URL                 | Представление | Описание                    |
|---------------------|---------------|-----------------------------|
| `/cart/`            | `cart`        | Отображение корзины         |
| `/cart/add/<id>`    | `cart_add`    | Добавление товара в корзину |
| `/cart/remove/<id>` | `cart_remove` | Удаление товара из корзины  |

## Особенности реализации

### Корзина для авторизованных пользователей

Корзина хранится в базе данных в модели `Cart`.

### Корзина для анонимных пользователей

Корзина хранится в сессии Django (`request.session['cart_items']`).
