# Приложение `security`

## Назначение

Приложение для обеспечения безопасности, мониторинга и логирования действий пользователей.

## Основные функции

- **Логирование авторизаций** — отслеживание входов и выходов пользователей
- **Управление устройствами** — парсинг и хранение информации об устройствах
- **Уведомления безопасности** — система уведомлений о важных событиях
- **Определение страны по IP** — интеграция с GeoIP

## Структура

```
security/
├── models/
│   ├── auth.py        # SecurityDevice, SecurityAuthLog
│   └── notice.py      # SecurityAction, SecurityNotice
├── signals.py         # Автоматическое логирование
├── utils.py           # Утилиты безопасности
└── admin/             # Настройки админ-панели
```

## Автоматическое логирование

Приложение автоматически логирует:

- Вход пользователя (`log_user_login`)
- Выход пользователя (`log_user_logout`)

Логирование происходит через Django signals.
