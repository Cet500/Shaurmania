# Сервисы приложения `geodata`

## `Geocoder`

Сервис для геокодирования адресов (определение координат по адресу).

### Основные функции

- Определение координат (широта, долгота) по адресу
- Нормализация адресов
- Интеграция с внешними API геокодирования
- Ограничение количества запросов

### Использование

```python
from geodata.services import Geocoder

geocoder = Geocoder()
result = geocoder.geocode('Москва, Красная площадь, 1')
# Возвращает координаты и нормализованный адрес
```

### Ограничения

Количество запросов ограничено настройкой `GEO_CODER_LIMIT` в `.env`.

### Статистика

Модель `GeocodeStat` хранит статистику использования геокодера:

| Поле               | Тип          | Описание            |
|--------------------|--------------|---------------------|
| `date`             | DateField    | Дата                |
| `requests_count`   | IntegerField | Количество запросов |
| `successful_count` | IntegerField | Успешных запросов   |
| `failed_count`     | IntegerField | Неудачных запросов  |
