# Приложение `geodata`

## Назначение

Приложение для работы с географическими данными: страны, регионы, города, улицы, адреса и часовые пояса.

## Основные функции

- **Географическая иерархия** — страны → регионы → города → улицы
- **Адреса** — полные адреса с координатами
- **Геокодирование** — определение координат по адресу
- **Часовые пояса** — управление часовыми поясами

## Структура

```
geodata/
├── models/
│   ├── geo.py          # Географическая иерархия
│   ├── address.py      # Адреса
│   └── timezones.py    # Часовые пояса
├── services/
│   └── geocoder.py     # Сервис геокодирования
├── admin/              # Настройки админ-панели
├── management/
│   └── commands/       # Management команды
└── fixtures/           # Тестовые данные
```

## Management команды

| Команда            | Описание                   |
|--------------------|----------------------------|
| `create_timezones` | Создание часовых поясов    |
| `download_geodata` | Загрузка геоданных         |
| `load_states_data` | Загрузка данных о регионах |
| `load_cities_data` | Загрузка данных о городах  |
| `base_geo_import`  | Базовый класс для импорта  |
