# Приложение `api`

## Назначение

REST API для административных функций и интеграций.

## Основные функции

- **Генерация тестовых данных** — создание тестовых объектов через Factory Boy
- **Геокодирование** — определение страны по IP-адресу

## Структура

```
api/
├── views.py           # API представления
├── urls.py            # URL-маршруты
└── apps.py            # Конфигурация приложения
```

## Документация

- [Эндпоинты API](endpoints.md) — полная документация всех эндпоинтов

## Аутентификация

Большинство эндпоинтов требуют:

1. Аутентификации пользователя (login required)
2. Прав суперпользователя (is_superuser=True)

## Связи с другими приложениями

- **`main`** — использует фабрики для генерации данных
- **`geodata`** — использует утилиты для геокодирования
