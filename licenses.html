Лицензия на продажу самой лучшей шаурмы от Льва Троцкого.

1. Общие положения

Настоящая лицензия предоставляет право на использование товарного знака «Шаурмания» для обозначения деятельности по продаже шаурмы с доставкой для мировой революции.

2. Права и обязанности сторон.

2.1. Владелец товарного знака «Шаурмания» предоставляет пользователю право использовать товарный знак в рамках, определённых настоящим документом.

2.2. Пользователь обязуется соблюдать правила использования товарного знака, указанные в настоящем документе, и не нарушать права владельца товарного знака.

3. Условия использования товарного знака.

3.1. Товарный знак «Шаурмания» может быть использован пользователем исключительно в рамках деятельности по продаже шаурмы с доставкой.

3.2. Использование товарного знака должно осуществляться в соответствии с законодательством Советского союза и международными нормами авторского права.

4. Ответственность сторон.

4.1. В случае нарушения условий настоящего документа пользователь несёт ответственность в соответствии с действующим законодательством Советского Союза.

4.2. В случае нарушения прав владельца товарного знака «Шаурмания» владелец имеет право требовать возмещения убытков и применения мер ответственности, предусмотренных законодательством.

5. Срок действия лицензии.

Лицензия действует бессрочно, пока существует деятельность по продаже шаурмы с доставкой под товарным знаком «Шаурмания».

6. Заключительные положения.

6.1. Все изменения и дополнения к настоящему документу должны быть оформлены в письменном виде и подписаны обеими сторонами.

6.2. Настоящий документ составлен в двух экземплярах, по одному для каждой стороны.
